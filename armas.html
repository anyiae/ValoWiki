Armas de Valorant

Guia de armas

Listamos todas las armas disponibles en Valorant, el FPS de Riot Games. Te enseñamos su velocidad de carga, tipo y una breve descripción de cada una.



Armas

ARES



  • TIPO // ARMAS PESADAS.

  • TIEMPO DE RECARGA// 3.25 SEGUNDOS.

  • El gran cargador de la Ares significa que se le da de maravilla el fuego continuo
    • para infligir grandes cantidades de daño a grupos de enemigos.

    BUCKY



  • TIPO // ESCOPETA.

  • TIEMPO DE RECARGA// 2.5 SEGUNDOS.

  • Pesada pero estable,tiene dos tipos de disparo: El disparo principal que ayuda a
    • vigilar la esquinas en espacios reducidos e ideal para cargar cuerpo a cuerpo.El modo de disparo
    alternativo es preciso a media distancia.

    BULLDOG



  • TIPO // RIFLE.

  • TIEMPO DE RECARGA // 2.5 SEGUNDOS.

  • Este rifle se trata de una apuesta segura.El modo alternativo tiene una mira
    • que permite disparar rafagas cortas con precisión a media y larga distancia.

    CLASSIC



  • TIPO // ARMA DE MANO.

  • TIEMPO DE RECARGA // 1.75 SEGUNDOS.

  • El modo disparo principal es preciso cuando estas quieto,tambien dispone de un modo
    • de disparo en ráfaga para los encuentros cuerpo a cuerpo.

    FRENZY



  • TIPO // ARMA DE MANO.

  • TIEMPO DE RECARGA // 1.5 SEGUNDOS.

  • Una ametralladora ligera que destaca a la hora de disparar en movimiento.Tiene una muy alta
    • velocidad de disparo,pero es difícil apuntar,asi que su uso a media distancia es ideal.

    GHOST



  • TIPO // ARMA DE MANO.

  • TIEMPO DE RECARGA // 1.5 SEGUNDOS.

  • La Ghost cuenta con un amplio cargador.Si el objetivo esta lejos deberas controlar la
    • velocidad de disparo.Pulsa el gatillo rapido cuando un enemigo este cerca.

    GUARDIAN



  • TIPO // RIFLE.

  • TIEMPO DE RECARGA // 2.5 SEGUNDOS.

  • Un rifle para tiradores por excelencia.Es un rifle pesado y cuenta con menos movilidad
    • que las demas armas,pero tambien es preciso y potente.Perfecto para media y larga distancia.

    JUDGE



  • TIPO // ESCOPETA.

  • TIEMPO DE RECARGA // 2.2 SEGUNDOS.

  • Es un arma estable,pero se vuelve vólatil si se dispara muy rapido.El modo de disparo
    • principal es perfecto para cortas distancias; para acercarse a enemigos a mas de un metro
    de distancia,se tiene que apuntar con calma y precisión.

    MARSHAL



  • TIPO // FUSIL.

  • TIEMPO DE RECARGA // 2.5 SEGUNDOS.

  • Es un fusil francotirador ligero con un zoom que mantendra a los enemigos a raya.Su baja
    • velocidad de disparo garantiza que, si no acierta al enemigo,quedaras expuesto.

    NAVAJA TÁCTICA



  • TIPO // CUERPO A CUERPO.

  • TIEMPO DE RECARGA // NULO.

  • Si te quedas sin balas, siemrpe contaras con esta navaja.Con ella correras y destruiras
    • objetos mas rápido y acabarás enemigos por la espalda de una cuchillada en su modo alternativo.

    ODIN



  • TIPO // ARMA PESADA.

  • TIEMPO DE RECARGA // 5 SEGUNDOS.

  • Dispara de forma continua e inflige mucho daño con una estabilidad sorprendente.Cubre de
    • balas a los enemigos en espacios reducidos o recurre al modo alternativo para convertirla en una torreta.

    OPERATOR



  • TIPO // FUSIL.

  • TIEMPO DE RECARGA // 3.7 SEGUNDOS.

  • Es un fusil de francotirador de cerrojo muy potente con zoom dual.Cuenta con movilidad
    • nula,pero sus balas derribaran a cualquiera de un solo impacto.

    PHANTOM



  • TIPO // RIFLE.

  • TIEMPO DE RECARGA // 2.5 SEGUNDOS.

  • Su modo automático es perfecto para los encuentros en espacios reducidos,mientras que en otra
    • situación sus ráfagas cortas son la mejor opción.Es mas precisa cuando se dispara quieto.

    SHERIFF



  • TIPO // ARMA DE MANO.

  • TIEMPO DE RECARGA // 2.25 SEGUNDOS.

  • Sus poderosos disparos causan un retroceso pronunciado y dominar esta arma requiere dedicación.
    • Si la empuñas con precisión, tus enemigos correrán al verte.

    SHORTY



  • TIPO // ARMA DE MANO.

  • TIEMPO DE RECARGA // 1.75 SEGUNDOS.

  • Es una escopeta ligera de barril corto que resulta letal cuerpo a cuerpo pero
    • solo puede dispararse dos veces antes de recargar.Complementa muy bien las armas de larga distancia.

    SPECTRE



  • TIPO // SUBFUSIL.

  • TIEMPO DE RECARGA // 2.25 SEGUNDOS.

  • Es un arma todoterreno con estadísticas de daño, velocidad de disparo y
    • precisión muy equilibradas tanto a distancias cortas como medias.Acecha en las esquinas de todos
    los mapas y, si tienes buen pulso, te permitirá acabar con tus enemigos desde lejos.

    STINGER



  • TIPO // SUBFUSIL.

  • TIEMPO DE RECARGA // 2.25 SEGUNDOS.

  • Este subfusil es más potente a media y larga distancia que el resto,
    • pero para ello sacrifica parte de su velocidad de disparo y movilidad.El cargador de 20 balas
    puede vaciarse en un abrir y cerrar de ojos si te dejas llevar por el retroceso,pero tambien te permitira
    acertar disparos letales a distancias medias con ayuda de la mira y un disparo controlado.

    VANDAL



  • TIPO // RIFLE.

  • TIEMPO DE RECARGA // 2.5 SEGUNDOS.

  • El Vandal inflige mucho daño a larga distancia y es ideal para aquellos que se centran en acertar tiros a la cabeza.
    • Sin embargo, va perdiendo estabilidad con el tiempo.
